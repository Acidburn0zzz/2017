Pulse 0.1.2 Release Notes

28th November, 2014 — George MacRorie

Pulse 0.1.2 is out.

This is a maintenance release that incorporates the updates in Syncthing v0.10.8 by The Syncthing Authors.

Here’s the full list of commits in this release.

There are changes and additions to the build process, symlink support, removal of insecure cipher suites and an autocomplete feature to the Add Device tab (currently only works in Chrome).

We’ve also opened account sign ups on source.ind.ie, so if you notice any issues, you can raise them on the Pulse Issues site
